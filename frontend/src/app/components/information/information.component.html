Cookies & Sessions

Cloudflare Services

Trader does not read, write, or interact with any cookies, local sessions, or local storage. However KPNC Technology uses Cloudflare services to provide users with the best experience and to protect our own infrastructure. Cloudflare sits in between your client and our origin servers and may read, write, or interact with their own cookies, local sessions, or local storage for a variety of reasons.

Market Data

Tiingo API, Metals Live API

The underlying stock, forex, and crypto information being displayed in Trader is provided by Tiingo and IEX. Information on precious metals and other commodities are from Metals Live.

If you are interested in market data see Tiingo API and Metals Live API for more information...

Backend Technologies

Cloudflare Workers, Tiingo API, Metals Live API

The backend of Trader uses Tiingo API (see above) to get current information on stock markets and Metals Live API for public information on precious metals and other commodities. Currently Trader has access to six diverse indexes including the S&P 500, Nasdaq 100, Dow Jones Industrial Average, Forex, Crypto, and Metals. All backend requests are routed through Cloudflare Workers, a relatively new serverless services by Cloudflare that allows for low latency requests that power the entire site.

See more information at Cloudflare Workers, Tiingo API, and Metals Live API...

Frontend Technologies

Angular, Lightweight Charts

The frontend of Trader uses the Angular framework for a powerfull and performant experience. However, data visualization is arguably just as important in trading so Trader uses Lightweight Charts from TradingView. Lightweight Charts provides fast and easy financial based charts that are highly effective for end users.

See more information at Angular and Chart.js...
