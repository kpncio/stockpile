Cookies & Sessions

Cloudflare Services

Trader does not read, write, or interact with any cookies, local sessions, or local storage. However KPNC Technology uses Cloudflare services to provide users with the best experience and to protect our own infrastructure. Cloudflare sits in between your client and our origin servers and may read, write, or interact with their own cookies, local sessions, or local storage for a variety of reasons.

Backend Technologies

Cloudflare Workers, IEX Cloud API

The backend of Trader uses IEX Cloud (see above) to get current information on stock markets. Currently Trader only supports the NDX (NASDAQ-100 Index) which includes the top 100 NASDAQ companies like Google, Microsoft, Apple, etc. All backend requests are routed through Cloudflare Workers, a relatively new serverless services by Cloudflare that allows for low latency requests that power the entire site.

See more information at Cloudflare Workers and IEX Cloud...

Frontend Technologies

Angular, ECharts, Chart.js

The frontend of Trader uses the Angular framework for a powerfull and performant experience. However data visualization is arguably just as important in Trader's use case. Trader uses two different chart libraries with accompanying wrappers, ECharts with ngx-echarts and Chart.js with ng2-charts. ECharts is blazingly fast and is able to allow for the quick browsing capabilities of the exchange page. Chart.js is beautiful with easy curve, gradient, and directive support for the multiple detailed views within the quote pages.

See more information at Angular, ECharts, and Chart.js...
